## ✨ Features

- 🎯 **Custom Cursor** — Gold animated cursor with smooth follower effect
- 🌌 **Animated Hero** — Floating orbs, grid background, gradient typography
- 📊 **Live Counters** — Stats animate when scrolled into view
- 🃏 **Services Grid** — 6 service cards with hover gold-line reveal
- 👥 **Team Section** — Avatar cards for all 5 team members
- 🔄 **Process Flow** — 4-step process with golden connecting line
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- ⚡ **Zero Dependencies** — Pure HTML + CSS + JavaScript, no frameworks

```bash

```
